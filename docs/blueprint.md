# **App Name**: KodNest Premium Build System

## Core Features:

- Global Layout Structure: Ensures every page follows a consistent layout: Top Bar -> Context Header -> [Primary Workspace + Secondary Panel] -> Proof Footer.
- Consistent Spacing System: Implements a strict spacing scale of 8px, 16px, 24px, 40px, 64px to maintain visual coherence and whitespace balance.
- Top Bar Design: Specifies the elements and layout for the top bar, including project name (left), progress indicator (center), and status badge (right).
- Context Header Design: Outlines the design for the context header, featuring a large serif headline, concise subtext, and clear purpose statement.
- Primary Workspace and Secondary Panel Structure: Defines the structure of the primary workspace (70% width for main interaction) and secondary panel (30% width for step explanation and copyable prompt).
- Proof Footer with Checklist: Creates a persistent proof footer with a checklist (UI Built, Logic Working, Test Passed, Deployed) requiring user proof input for each checkbox.
- Error Handling with AI Assistant Tool: AI tool will guide user in case of failure states

## Style Guidelines:

- Background color: Off-white (#F7F6F3) for a calm and clean feel.
- Primary text color: Dark gray (#111111) for high readability.
- Accent color: Deep red (#8B0000) for primary buttons and important actions.
- Headings: 'Literata' serif font for a confident and legible headline.
- Body: 'Inter' sans-serif font, 16-18px, for clean and readable text blocks.
- Consistent use of spacing based on an 8px grid (8px, 16px, 24px, 40px, 64px).
- Subtle transitions of 150-200ms with ease-in-out timing.
